# GCD (Greatest Common Divisor)

This crate provides two GCD implementations for comparison:

- Euclidean algorithm (modulo-based)
- Binary GCD (Stein's algorithm)

## Complexity

- Euclidean: O(log min(a, b))
- Binary GCD: O(log min(a, b))
